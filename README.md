#### 软件架构
软件架构说明
功能点：
    模拟商城，完整的购物流程，和对项目的运维，有各项的监控指标和运维指标。
技术点：
       核心技术为springcloud+vue两个全家桶实现
	
	核心框架：springcloud Edgware全家桶
	安全框架：Spring Security Oauth2
	持久层框架：MyBatisPlus
	数据库连接池：Alibaba Druid
	前端框架：Vue全家桶以及相关组件
